# Phase 1 Scope — SmartLife Flexi Growth OS

## Included in Phase 1

### Self-Serve Onboarding
- 5-step journey: Segment → Goal → Plan Inputs → Projection → Personalised Plan
- Live projection calculation (compound interest, 10% indicative)
- Personalised copy for 8 saver segments × 15 goals
- No NIN, no phone, no email in demo

### Staff Assist
- Session form for advisors
- Script generator with objection handling (9 objection types)
- CRM stage suggestion
- Follow-up task logging

### Projection Calculator
- Interactive calculator page
- Frequency comparison table (daily/weekly/monthly/annually)
- Goal milestone markers
- Debounced live API calls

### Payment Simulation
- Demo checkout with Pesapal branding
- Simulated success flow
- No real payment gateway connected

### Analytics
- GTM-PZRV3MQL container
- 10 analytics event types
- Param sanitisation (no PII in events)
- Dropoff detection

### Support
- Support request form (demo)
- FAQ section
- Staff escalation option

## NOT Included in Phase 1
- Real NSSF member authentication
- NIN verification
- Real payment processing
- SMS delivery
- Email delivery
- Member portal
- Employer deduction workflow
- Regulatory compliance sign-off
